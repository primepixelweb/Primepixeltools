What is a Random Number Generator?

A random number generator (RNG) is a tool that produces numbers randomly within a specified range. It is widely used in statistics, games, simulations, programming, and research.

How to Use the Random Number Generator

1. Enter the minimum number.
2. Enter the maximum number.
3. Click the generate button.
4. A random number will appear instantly.

Example

If the minimum number is 1 and the maximum number is 100, the generator may return any number such as 42.
